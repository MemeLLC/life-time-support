# 出力テンプレート

Phase 3 でレポート化する際にこのテンプレートをベースに使用する。

```markdown
# [調査テーマ] 業界全体像（Industry Landscape）

調査日: YYYY-MM-DD

---

## 業界概要
- 業界の定義と範囲
- 歴史的背景
- 業界の成熟度

## 用語集
| 用語 | 定義 | 補足 |
|------|------|------|
（最低20〜30語）

## ビジネスモデル
- 主要なビジネスモデル
- 収益構造とお金の流れ
- 価格帯・利益率の水準

## バリューチェーンとキープレイヤー
- バリューチェーン（川上→川下の構造）
- 主要プレイヤーとシェア
- 競争環境の特徴・力関係

## 参照ソース
| # | サイト名 | URL |
|---|----------|-----|
```
