# 競合分析 出力テンプレート

競合ごとにファイルを分割する。実施しなかった調査項目のセクションはテンプレートから削除する。情報が得られなかった項目は「未確認」と記載し、空欄にしない。スクリーンショットは `![説明](相対パス)` 形式で埋め込み、マークダウンプレビューで直接表示できるようにする。

## ファイル構成

```
competitor-analysis.md              # メイン（要約・競合リスト・横断比較・示唆）
competitor-analysis/
├── [競合名1].md                    # 競合1の詳細（基本情報・広告・SNS等）
├── [競合名2].md                    # 競合2の詳細
├── [競合名3].md                    # 競合3の詳細
├── psi-[競合名]-mobile.json        # PSI生データ
└── [競合名]-ad-01.png              # スクリーンショット
```

メインファイルから各社の詳細ファイルへ相対リンクで参照する。
