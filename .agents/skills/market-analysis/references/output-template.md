# 出力テンプレート

Phase 3 でレポート化する際にこのテンプレートをベースに使用する。

```markdown
# [調査テーマ] 市場調査（Market Analysis）

調査日: YYYY-MM-DD

---

## 要約
[3〜5行の要約]

## 市場規模
- 市場規模（金額ベース）
- 市場規模（数量ベース）
- 成長率（過去・予測）
- セグメント別内訳
- ソース間の数値比較

## 市場構造
- 主要プレイヤー（上位3〜5社）とシェア
- 流通チャネル
- 参入障壁の有無と程度
- 市場の成熟度
- 市場の集中度

## 価格調査
- 市場の価格帯（最低〜最高、中央値）
- 主要競合の料金比較
- 採用されている料金モデルの種類
- 価格戦略の傾向

## 顧客調査
### 属性データ
- 年齢層・性別・職種・年収・居住地域・学歴・世帯構成
### 行動・心理
- 購買動機・課題・情報収集行動・重視する選定基準
### ペルソナ像
- 典型的なユーザー像（データに基づく）

## 示唆・考察
[調査結果から導かれる示唆]

## 参照ソース
| # | サイト名 | URL |
|---|----------|-----|
```
