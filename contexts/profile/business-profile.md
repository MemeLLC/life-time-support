## 事業内容

### サービス・プロジェクト概要

- **サービス名**: ライフタイムサポート（LTS）
- **一言での説明**: 首都圏の新築マンション購入者向けインテリアオプション工事
- **詳細な説明**: 新築マンション購入者に対して、フロアコーティング、エコカラット、家具・照明取付、壁面クロスなど多種多様なオプション工事をワンストップで提供する。デベロッパーのオプション販売会よりも低価格で高品質な施工を行い、内覧会同行サービス（200項目チェック）も提供している。

### サービスメニュー（17種類以上）

エコカラット / フロアコーティング / 水周りコーティング / クロスコーティング / 白木コーティング / オーダーミラー / バルコニータイル / オーダー家具 / 輸入クロス / キャットウォーク / オーダーカーテン / ピクチャーレール / 窓ガラスフィルム / ビルトイン食洗機 / ビルトインオーブン / IHクッキングヒーター / レンジフードフィルター / カップボード / 可動棚 / ダウンライト / 人感センサー 等

### ターゲット

- **顧客タイプ**: B2C
- **主な顧客層**: 首都圏の新築マンション購入者（ファミリー層が中心と推定）
- **展開地域**: 関東全域（東京都、神奈川県、埼玉県、千葉県、群馬県、茨城県、栃木県）+ 名古屋（愛知県）。首都圏がメイン

### ビジネスモデル

- **収益モデル**: 工事請負（都度受注）
- **料金体系**: 工事内容ごとの見積もり制（顧客ごとにカスタム）
- **平均受注単価**: 100万〜150万円
- **月間受注件数**: 100件以上
- **主な収益源**: オプション工事の施工費
- **料金例（エコカラット）**: 1平米あたり9,600円〜24,000円（施工費込み）

### サービスの特徴

- **独自の強み（USP）**:
  - ワンストップ対応（複数工事を一括で依頼可能、立会い2回のみ、1週間〜10日で完了）
  - デベロッパーのオプション販売会より低価格（中間マージン排除）
  - 内覧会同行サービス（建築士が200項目をチェック）
  - マイスター制度・月次勉強会による高い施工品質
  - 15年以上の施工経験
  - 施工後もLINEサポート対応
- **主な実績**:
  - 施工実績 20,000件以上
  - 年間600件以上のマンション工事
  - 月間受注100件以上
  - 2023年度 SMB Excellent企業賞 受賞
- **経営理念**: 「ライフタイムサポートでよかった」その一言の為に
- **保証**: シリコンコーティング15年、UV・ガラスコーティング20年

### 事業フェーズ

- **現在のフェーズ**: 安定期（さらなる売上拡大・効率化を目指す段階）
- **売上高**: 約4億8,000万円（リフォームオンライン記事）〜5億円前後（ユーザー情報）

### ブランドアイデンティティ

- **キャッチコピー**: 「マイホームにプラス1％の感動を」

### 現状の課題

- 数値が見える化されていない（全体的な計測基盤の不足）
- オーガニックが強いのは競合が弱いからであり、自社のマーケティング力によるものではない
- HPは導線が弱い（コンテンツ過多、ラヴィゴット制作）
- HPのmeta description・OGPが未設定
- HPのモバイルパフォーマンスが深刻に低い（Performance: 29）
- 広告予算は月最大100万円と限定的
- CVR改善の余地がある

### 目標

- **短期（3〜6ヶ月）**: 年商5億円 → 8億円への引き上げ
- **中長期**: 年商15億円
- **長期ビジョン**: 2046年までに売上100億円・従業員200名

### 戦略の方向性

1. 全体的なCVR向上
2. オーガニックだけでなく広告でも安定した売上を確保（リスクヘッジ）
3. 数値の見える化（計測基盤の構築）
